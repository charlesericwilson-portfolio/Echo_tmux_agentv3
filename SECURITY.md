# Security Policy

We take security seriously. Please report vulnerabilities via email: charles.eric.wilson@gmail.com or open an issue if public.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 5.1.x   | :white_check_mark: |
| 5.0.x   | :x:                |
| 4.0.x   | :white_check_mark: |
| < 4.0   | :x:                |

## Reporting a Vulnerability

Please report via email or issues. We will respond within 1-3 business days.
